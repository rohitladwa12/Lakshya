# Dynamic Resume Builder – AI Powered

## Overview
Student-facing resume builder page. Allows AI-driven generation from profile data or PDF upload.

## Features
- **Generate Resume**: One-click AI generation via OpenAI.
- **Upload PDF**: Upload and preview PDF resumes.
- **Edit Mode**: Visual and HTML editors for fine-tuning.
- **Download/Print**: Export as A4 PDF or print directly.

## Files
- `public/student/resume_builder.php`: Main page.
- `openai_config.php`: OpenAI API settings.
- `public/student/includes/config.php`: Page-specific config.
